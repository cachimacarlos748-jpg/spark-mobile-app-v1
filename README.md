# SparkIA - Aplicación Móvil de IA

SparkIA es una aplicación móvil elegante y potente que te permite chatear con Spark, un asistente de inteligencia artificial basado en Gemini. Diseñada con Material Design 3 para una experiencia premium.

## Características

- Chat con Spark (IA avanzada)
- Múltiples modelos: Flash, Flash-Lite, Pro
- Razonamiento avanzado (estándar y extendido)
- Generación de imágenes
- Menú de opciones (Fotos, Cámara, Archivos, Imágenes, Música, Canvas, Deep Research)
- Modo offline
- Tema oscuro/claro automático
- Autenticación con Google
- Sincronización en la nube

## Publicar en Google Play Store

Ver archivo `PUBLICAR_EN_GOOGLE_PLAY.md` para instrucciones completas.

## Requisitos

- Flutter 3.24.0+
- Android SDK 34+
- Java 11+
- Gemini API Key

## Instalación

```bash
flutter pub get
flutter run
```

## Compilar APK

```bash
flutter build apk --release
```

## Estructura del proyecto

El proyecto está organizado en carpetas por funcionalidad: screens (pantallas), widgets (componentes), services (servicios), providers (gestión de estado) y theme (temas).
